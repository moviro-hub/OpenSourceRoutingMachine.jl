Core API

Core types, configuration, and utilities.

Types

OpenSourceRoutingMachine.LatLon — Type
LatLon

A named tuple representing a latitude and longitude coordinate pair.

source
OpenSourceRoutingMachine.OSRM — Type
OSRM(base_path::String)

High-level handle for querying an OSRM dataset located at base_path.

source
OpenSourceRoutingMachine.OSRMConfig — Type
OSRMConfig(base_path::String)

Low-level configuration handle for OSRM; most callers will use OSRM(base_path) instead of constructing this directly.

source

Configuration Functions

OpenSourceRoutingMachine.set_algorithm! — Method
set_algorithm!(config::OSRMConfig, algorithm)

Force a specific routing algorithm for the given configuration instead of letting it be inferred from dataset files on disk.

source
OpenSourceRoutingMachine.set_dataset_name! — Method
set_dataset_name!(config::OSRMConfig, dataset_name)

Select which named dataset OSRM should attach to when running in shared-memory mode (or clear it by passing nothing).

source
OpenSourceRoutingMachine.set_default_radius! — Method
set_default_radius!(config::OSRMConfig, radius)

Set the default snapping radius used when no per-coordinate radius is provided.

source
OpenSourceRoutingMachine.set_max_alternatives! — Method
set_max_alternatives!(config::OSRMConfig, max_alternatives)

Configure the global upper bound on how many route alternatives OSRM may return.

source
OpenSourceRoutingMachine.set_max_locations_distance_table! — Method
set_max_locations_distance_table!(config::OSRMConfig, max_locations)

Configure the maximum number of locations accepted for Table (matrix) queries.

source
OpenSourceRoutingMachine.set_max_locations_map_matching! — Method
set_max_locations_map_matching!(config::OSRMConfig, max_locations)

Configure how many coordinates OSRM will accept for Map Matching requests.

source
OpenSourceRoutingMachine.set_max_locations_trip! — Method
set_max_locations_trip!(config::OSRMConfig, max_locations)

Configure the maximum number of locations OSRM will accept for Trip queries.

source
OpenSourceRoutingMachine.set_max_radius_map_matching! — Method
set_max_radius_map_matching!(config::OSRMConfig, radius)

Set the maximum search radius OSRM will use when snapping points for Map Matching queries.

source
OpenSourceRoutingMachine.set_max_results_nearest! — Method
set_max_results_nearest!(config::OSRMConfig, max_results)

Limit how many nearest candidates OSRM should return for Nearest queries.

source
OpenSourceRoutingMachine.set_use_mmap! — Method
set_use_mmap!(config::OSRMConfig, use_mmap)

Toggle whether OSRM should memory-map datasets instead of loading them fully into RAM.

source
OpenSourceRoutingMachine.set_use_shared_memory! — Method
set_use_shared_memory!(config::OSRMConfig, use_shared_memory)

Control whether OSRM should attach to a shared-memory region populated by osrm-datastore for dataset access.

source
